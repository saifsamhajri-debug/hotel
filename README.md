# نظام إدارة الفندق

نظام متكامل لإدارة الفنادق مبني باستخدام Python وإطار العمل Flask.

## الميزات

- **نظام تسجيل الدخول للمستخدمين**:
  - نوعان من المستخدمين: موظف الاستقبال (Admin) والنزلاء (Guests)
  - دعم لتسجيل وتوثيق الدخول، وتغيير كلمة المرور

- **إدارة الغرف**:
  - عرض قائمة الغرف المتاحة والمشغولة
  - إضافة، تعديل، حذف الغرف
  - تحديد مواصفات كل غرفة (رقم، نوع، سعر، حالة)

- **الحجوزات**:
  - إمكانية حجز غرفة لتاريخ معين
  - التحقق من توفر الغرف في الفترة المطلوبة
  - عرض الحجز حسب المستخدم أو حسب التواريخ
  - **تسجيل أوقات الدخول والخروج الفعلية** من ساعة النظام
  - عرض أوقات تسجيل الدخول والخروج مع بيانات العميل

- **إدارة العملاء**:
  - إضافة بيانات العميل (اسم، رقم الهوية، هاتف، بريد)
  - ربط كل عميل بالحجز الذي قام به

- **الخدمات الإضافية**:
  - إضافة خدمات للحجز (وجبات، غسيل ملابس، سبا، إلخ)
  - تحديد الكمية والسعر لكل خدمة
  - حساب التكلفة الإجمالية للخدمات

- **نظام الفواتير**:
  - عرض فاتورة تفصيلية لكل حجز
  - تشمل الفاتورة: تكلفة الإقامة + الخدمات الإضافية
  - **طباعة الفاتورة** مباشرة من المتصفح
  - **تحميل الفاتورة كملف PDF**

- **تخصيص اسم الفندق والشعار**:
  - إمكانية تغيير اسم الفندق من ملف الإعدادات
  - عرض اسم الفندق في القائمة الرئيسية والفواتير

- **لوحة تحكم للمسؤول**:
  - رؤية إجمالية للحجوزات، الغرف، والإيرادات

## متطلبات التشغيل

- Python 3.8+
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-WTF
- Flask-Migrate
- وغيرها من المكتبات (مذكورة في ملف requirements.txt)

## طريقة التثبيت

1. قم بإنشاء بيئة افتراضية:
   ```
   python -m venv venv
   ```

2. قم بتفعيل البيئة الافتراضية:
   - في Windows:
     ```
     venv\Scripts\activate
     ```
   - في Linux/Mac:
     ```
     source venv/bin/activate
     ```

3. قم بتثبيت المكتبات المطلوبة:
   ```
   pip install -r requirements.txt
   ```

4. قم بتهيئة قاعدة البيانات وإضافة بيانات تجريبية:
   ```
   python init_db.py
   ```

5. (اختياري) قم بتخصيص اسم الفندق:
   - انسخ ملف `.env.example` إلى `.env`
   - قم بتعديل قيمة `HOTEL_NAME` في ملف `.env`

6. قم بتشغيل التطبيق:
   ```
   python run.py
   ```

7. افتح المتصفح على العنوان:
   ```
   http://localhost:5000
   ```

## بيانات الدخول الافتراضية

- **حساب المسؤول**:

  - كلمة المرور: admin123

- **حساب النزيل**:
  - اسم المستخدم: guest
  - كلمة المرور: guest123
